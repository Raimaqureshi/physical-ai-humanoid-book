## Slide 8: Technical Architecture

-   **Frontend**: Docusaurus, React (User Interface, Content Rendering).
-   **Backend**: FastAPI (APIs for AI, Auth, Personalization, Content).
-   **Database**: Neon Serverless Postgres (User data, Metadata).
-   **Vector DB**: Qdrant Cloud (Text embeddings for RAG).
-   **AI**: OpenAI Agents / ChatKit SDK (LLM, Embeddings).
-   **Deployment**: GitHub Pages (Frontend CI/CD).
